FAQ

Bagaimana cara saya menerima aplikasinya setelah bayar?

Setelah pembayaran melalui Mayar berhasil, Anda mendapat akses unduh file aplikasi Premium (1 file HTML). Cukup buka filenya di browser — tidak perlu instalasi khusus.

Apakah ini langganan? Apa saya perlu bayar lagi tahun depan?

Tidak. Ini pembelian sekali bayar. File yang Anda unduh menjadi milik Anda dan bisa dipakai selamanya, tanpa biaya berulang.

Apakah kalkulator AK ini akurat 100%?

Kalkulator memakai nilai AK Maksimal per kategori sesuai Kepmen No. 39/M/KEP/2026 — ini estimasi batas atas yang membantu perencanaan, bukan pengganti penilaian resmi asesor/reviewer PT Anda.

Data saya aman?

Ya. Semua data (profil, publikasi, riwayat skor) tersimpan di browser Anda sendiri (localStorage) dan tidak pernah dikirim ke server kami atau pihak lain.

Bisa dicoba dulu sebelum beli?

Bisa — coba versi Free terlebih dahulu untuk melihat cara kerja dasarnya, lalu upgrade kapan saja saat butuh fitur perencanaan yang lebih dalam.
