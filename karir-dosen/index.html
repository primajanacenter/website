FAQ

Apakah data saya dikirim ke server?

Tidak. Seluruh perhitungan berjalan di browser Anda sendiri, dan data tersimpan lokal di perangkat (localStorage). Tidak ada data yang dikirim ke server kami atau pihak lain.

Apakah ini alat resmi dari Kemdiktisaintek/Ditjen Dikti?

Bukan. Ini alat bantu simulasi independen berdasarkan Kepmen No. 39/M/KEP/2026. Keputusan resmi kenaikan jabatan tetap melalui SISTER dan unit SDM perguruan tinggi Anda.

Setelah beli Premium, bagaimana saya menerima filenya?

Setelah pembayaran melalui Mayar berhasil, Anda akan menerima file aplikasi Premium untuk diunduh langsung — tidak perlu instalasi, cukup buka filenya di browser.

Apakah perhitungan AK ini menggantikan penilaian asesor?

Tidak. Kalkulator AK memakai nilai maksimal per kategori sebagai estimasi batas atas — nilai final tetap ditentukan asesor/reviewer PT berdasarkan kualitas dan kelengkapan bukti kinerja Anda.

Bisa dipakai di HP?

Bisa. Tampilan menyesuaikan otomatis ke layar HP. Fitur "install sebagai aplikasi" tersedia untuk versi Premium, tapi hanya aktif jika file dibuka lewat situs web (https://) — kalau Anda hanya membuka file hasil unduhan langsung dari folder, aplikasi tetap berfungsi penuh, hanya saja tanpa opsi install tersebut.
